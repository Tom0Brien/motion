Detailed Description

template<typename Scalar, int nq>
struct tr::model::Model< Scalar, nq >

A motion model.

A motion model is a collection of links and joints that represents a robot.

Template Parameters
Scalar The scalar type for the model.
nq The number of configuration coordinates (number of degrees of freedom).
